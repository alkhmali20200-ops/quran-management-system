# رفع النظام على GitHub Pages مجاناً 🚀

## الخطوات:

### 1. إنشاء حساب GitHub
- اذهب إلى: https://github.com
- أنشئ حساب جديد (مجاني)

### 2. إنشاء مستودع جديد
- اضغط "New repository"
- اسم المستودع: `quran-management-system`
- اختر "Public"
- اضغط "Create repository"

### 3. رفع الملفات
- اضغط "uploading an existing file"
- اسحب جميع الملفات من مجلد `algmaha`:
  - index.html
  - styles.css
  - script.js
  - config.js
  - enhancements.js
  - quick-guide.html
  - test.html
  - README.md

### 4. تفعيل GitHub Pages
- اذهب إلى Settings في المستودع
- انزل إلى قسم "Pages"
- اختر "Deploy from a branch"
- اختر "main" branch
- اضغط "Save"

### 5. الحصول على الرابط
بعد دقائق قليلة، سيكون النظام متاح على:
**https://[اسم-المستخدم].github.io/quran-management-system**
