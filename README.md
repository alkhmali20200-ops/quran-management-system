# نظام إدارة جمعية تحفيظ القرآن الكريم 🕌

## نظرة عامة
نظام ويب شامل ومتطور لإدارة جمعيات تحفيظ القرآن الكريم، يوفر إدارة متكاملة للمعلمين والطلاب والحلقات مع متابعة دقيقة للحفظ والمراجعة، بالإضافة إلى أدوات تحليلية متقدمة وواجهة مستخدم عصرية.

## المميزات الرئيسية

### 🔐 نظام المستخدمين
- **ثلاثة أنواع من المستخدمين:**
  - إدارة: صلاحيات كاملة
  - معلمين: إدارة حلقاتهم وطلابهم
  - طلاب: عرض تقدمهم الشخصي

### 👥 إدارة المعلمين
- إضافة وتعديل بيانات المعلمين
- تتبع الحلقات المسؤول عنها كل معلم
- إحصائيات عدد الطلاب لكل معلم

### 🎓 إدارة الطلاب
- إضافة وتعديل بيانات الطلاب
- تصنيف حسب الجنس والحلقة
- متابعة مقدار الحفظ والمراجعة
- نظام تقدير من ممتاز إلى ضعيف

### 🕌 إدارة الحلقات
- إنشاء حلقات منفصلة للذكور والإناث
- تحديد أوقات الحلقات (بعد الصلوات)
- ربط كل حلقة بمسجد أو دار
- جدولة من الأحد إلى الأربعاء

### 📊 المتابعة والإحصائيات
- لوحة تحكم تفاعلية مع الرسوم البيانية
- نسب الحفظ والمراجعة
- إحصائيات الحضور
- تقارير التقدم

### 📅 التقويم والجدولة
- تقويم يومي للأنشطة
- جدولة الحلقات حسب الأوقات
- متابعة الأنشطة والفعاليات

### 🏆 نظام التميز
- عرض الطلاب المتميزين أسبوعياً
- تصنيف منفصل للذكور والإناث
- تحفيز الطلاب للتفوق

## كيفية الاستخدام

### 1. تشغيل النظام
```bash
# افتح ملف index.html في المتصفح مباشرة
# أو استخدم خادم محلي (مستحسن)
python -m http.server 8000

# ثم انتقل إلى
http://localhost:8000
```

### 🔗 روابط مهمة
- **الصفحة الرئيسية**: `index.html`
- **دليل الاستخدام السريع**: `quick-guide.html`
- **صفحة الاختبار**: `test.html`

### 2. تسجيل الدخول
استخدم الحسابات التجريبية التالية:

**حساب الإدارة:**
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`
- النوع: إدارة

**حساب المعلم:**
- اسم المستخدم: `teacher1`
- كلمة المرور: `teacher123`
- النوع: معلم

**حساب الطالب:**
- اسم المستخدم: `student1`
- كلمة المرور: `student123`
- النوع: طالب

### 3. الوظائف الأساسية

#### للإدارة:
- إضافة وإدارة المعلمين
- إضافة وإدارة الطلاب
- إنشاء وإدارة الحلقات
- مراقبة الإحصائيات العامة
- إدارة الأنشطة والفعاليات

#### للمعلمين:
- عرض طلاب حلقاتهم
- تسجيل الحضور
- متابعة تقدم الطلاب
- تحديث درجات الحفظ والمراجعة

#### للطلاب:
- عرض التقدم الشخصي
- مراجعة الدرجات والتقديرات
- متابعة الأنشطة

## التقنيات المستخدمة

### 🛠️ التقنيات الأساسية
- **HTML5**: هيكل الصفحات مع دعم RTL
- **CSS3**: تصميم متجاوب وعصري مع Grid و Flexbox
- **JavaScript ES6+**: منطق التطبيق والتفاعل
- **Chart.js**: رسوم بيانية تفاعلية
- **Font Awesome 6**: أيقونات حديثة
- **Google Fonts**: خطوط عربية (Cairo, Amiri)
- **Local Storage**: تخزين البيانات محلياً

### ✨ المميزات المتقدمة
- **نظام إشعارات متطور**: إشعارات منزلقة مع أنيميشن
- **بحث ذكي**: بحث سريع في جميع البيانات (Ctrl+K)
- **اختصارات لوحة المفاتيح**: تنقل سريع (Ctrl+1-9)
- **حفظ تلقائي**: حفظ البيانات كل 30 ثانية
- **تصدير البيانات**: تصدير CSV ونسخ احتياطية JSON
- **تحليلات متقدمة**: إحصائيات وتقارير مفصلة


## المميزات التقنية

### 🎨 تصميم متجاوب
- يعمل على جميع الأجهزة (كمبيوتر، تابلت، هاتف)

- ألوان وتصميم احترافي

- استخدام Local Storage لحفظ البيانات

- بيانات تجريبية جاهزة للاختبار

- رسوم بيانية تفاعلية للإحصائيات
- عرض نسب الحفظ والحضور
- تحديث تلقائي للبيانات

- نظام صلاحيات متدرج
- إخفاء/إظهار المحتوى حسب نوع المستخدم
- حماية البيانات الحساسة
